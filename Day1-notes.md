# Day 1–Machine Learning Basics

## What is Machine Learning
Machine Learning is a way for computers to learn patterns from data
instead of being explicitly programmed.

## Fake News Detection Problem
The goal is to classify news articles as Fake or Real based on text.

## Key Concepts Learned
- What does training a model mean
- Supervised learning
- Input (text) and output (label)
- Why text must be converted to numbers

## ML Pipeline
Text → Cleaning → Vectorization → Model → Prediction
